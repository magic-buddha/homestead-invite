01What is this?

It's the full Escape from Tarkov — same maps, guns, quests, traders, flea market, hideout — running on a private server I host. You install a separate modded copy (your real EFT stays untouched), connect to my box, and we raid together co-op. Every enemy is AI, but not the brain-dead vanilla kind: a stack of the best community mods makes bots fight, quest, and loot like actual players.

If you played live Tarkov years ago and quit — this is the version that respects your time. If you never played: it's the most intense loot-shooter ever made, and this is the friendliest possible way in.

  • 🛡️ Insurance actually returns your gear (~95%), near-instantly — but careful: enemy PMCs loot bodies, and whatever they extract with is gone forever. Death still costs something.
  • 🏆 Prestige (level 55+) is enabled — the endgame reset with progressive skill/mastery retention is there when you earn it.

07How raids work with the crew

The server runs a headless host — a dedicated game instance on my box that hosts every raid. Nobody's PC does the heavy lifting, and nobody has to stay till the end: die early, leave, your loot state is saved.

One of us clicks Host Raid → checks Use Headless Host → everyone joins from the lobby → host presses Start. Kill XP is shared across the group, so nobody falls behind.

08Install (~1 hour, mostly downloads)

Requirements: Windows, a legit EFT account (any edition), ~70 GB free disk.

  1. Install live EFT — official BSG launcher or Steam, both work fine. Let it download fully. (Files get copied; you won't play live.)
  2. Install SPT. Get the installer at forge.sp-tarkov.com/installer, put it in a new empty folder (e.g. C:\SPT\), run it. When it finishes, run SPT.Launcher.exe, make a temp profile, and start one solo raid to confirm it works. Version must be SPT 4.0.13 — if an installer offers newer, stop and ask me.
  3. Install Fika (the co-op layer). Download Fika-Installer.exe from github.com/project-fika/Fika-Installer, put it in your SPT folder, run → Install Fika. The main menu should then show FIKA 2.3.x | SPT 4.0.13 bottom-left.
  4. Drop in the mod pack. Download homestead-modpack-2026-07-05.zip (~350 MB) and extract it into your SPT folder so BepInEx and EscapeFromTarkov_Data merge. Say yes to all merges. Never add/update mods on your own — versions are pinned to the server.
  5. Connect. SPT.Launcher → Settings (gear) → Server URL → the address I send you (starts with https://). Create your character through the launcher — pick the "SPT Zero to Hero" edition when asked: you start with a knife and nothing else. Trust us on this one — the server is tuned generous (see the tuning section), so the broke-scavenger arc at the start is the best part of the game. Then launch and play. (Shared squad XP is already enforced server-side — nothing to configure.)
Your first raid will load slowly — it can sit at a low percentage for several minutes while ~120 MB of custom assets download from the server (one time only). Don't alt-tab, don't force quit. It will move.

09If something breaks

  • • Launcher won't connect → re-check the Server URL, must be https://, not http://.
  • • Infinite loading / crash on join → your versions or mods don't match. Check bottom-left version text; re-extract the pack.
  • • Anything weird → send me BepInEx\LogOutput.log from your SPT folder. My server logs the rest.
