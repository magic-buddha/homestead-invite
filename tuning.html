TUNING & SETTINGS

Part 1 is the server owner's manual — the main dials that shape difficulty, bots, and loot. Part 2 is for everyone — make the game look good and run well on your PC.

01Server tuning — the six main dials

Admin-only. WebConfig links work from the home LAN only (accept the self-signed certificate warning).

DialTool & whereWhat it controls
How MANY botsABPS WebConfig — https://192.168.4.25:6969/botplacementsystemStarting PMCs per map, wave sizes/timing, max alive at once, boss spawn odds. The #1 lever for "raids feel empty / too crowded" and for headless CPU load.
How GEARED bots areAPBS WebConfig — https://192.168.4.25:6969/progressivebotsystemBot equipment tiers and progression speed. Gear already scales with the squad's average level automatically.
How well bots FIGHTSAIN — F12 on the gaming PC, preset homesteadserverAim speed, accuracy, reaction time, hearing/vision. Global difficulty sliders live under Difficulty. Leave Personality → Assignment alone — those chances are tuned for ORBIT.
How bots BEHAVEORBIT — F12 on the gaming PCBot looting thresholds, scav loot chance, roaming, extraction pacing (when bots leave the raid).
LOOT & economySVM via Greed.exe on the gaming PC (preset homesteadsetup)Loose/container loot multipliers, flea unlock level, insurance, hideout speed, trader stock, extracts, raid time. The whole economy in one preset file.
CO-OP rulesfika.jsonc on the server boxRevive system (currently: 3 revives, 5-min bleedout, no items needed), friendly fire, shared quest progression, bot level scaling.

How changes reach the server

  • SAIN / ORBIT (F12 mods): tune on the gaming PC, close the game, then copy the changed files to the headless — SAIN: the whole BepInEx\plugins\SAIN\Presets\homesteadserver folder; ORBIT: BepInEx\config\com.chazut.orbit.cfg. Then restart the headless container.
  • SVM: save the preset in Greed, drop the JSON in the server's incoming/ folder, install server-side, restart the server container.
  • ABPS / APBS WebConfig: applies from the browser — no file copying.
  • Rule of thumb: server-mod change → restart fika-server; headless plugin/config change → restart fika-headless.
Never-break rules: versions are pinned as a set (SPT 4.0.13 / Fika 2.3.4 / Headless 1.4.14) — never update one layer alone. Never reinstall Questing Bots or Looting Bots (they fight ORBIT). Never add AI-limiter/bot-culling mods (they freeze ORBIT squads). Keep raid times near vanilla. One change at a time, then a test raid.
Health checks during a raid: press ~ → type debug t — keep Server FPS ≥ 30. If it dips, lower ABPS bot counts before blaming anything else.

02Player settings — performance vs. looks

Tarkov is CPU-hungry and mod-heavy. These are the dials that matter, in order of impact. Everything else is taste.

If you need more FPS (biggest wins first)

SettingWhereDo this
DLSS / FSRGraphicsRTX card → DLSS Quality, otherwise FSR 2.2 Quality — biggest free win, but DLSS causes visible ghosting/smearing for some people (moving bots, scopes). If that bothers you (it does the host), turn it off and run TAA High instead — the rest of this list still buys plenty.
ShadowsGraphicsMedium. Low looks bad; Ultra eats frames. Grass shadows OFF.
Overall visibilityGraphics1500–2000 m. Only snipers on Woods/Lighthouse benefit from more.
Object LODGraphics2.5 — the sweet spot. Higher is mostly CPU cost.
SSR (reflections)GraphicsOFF. Expensive, barely visible.
Auto RAM cleaner + Only use physical coresGame tabBoth ON.

If you want it prettier

SettingWhereDo this
Anti-aliasingGraphicsTAA High (DLSS replaces it if enabled).
TexturesGraphicsHigh if you have 8 GB+ VRAM.
HBAOGraphicsHigh — but our AO Replacer mod (GTAO) already improves this; try its F12 presets first.
SharpnessGraphics0.4–0.7. Above that it gets crunchy.
"Washed out" look?F12 → Smajlec's LightsThis mod recolors the game — if it's too bright/flat for you, lower its intensity or switch its preset. Tune this before touching PostFX.
PostFXPostFX tabStart OFF. If you want punch: Contrast +5–10, Color grading to taste, leave Brightness alone. PostFX costs a few FPS.

Game & controls worth setting once

  • FOV 65–75 — our FOV Fix mod keeps scopes correct at high FOV.
  • Head bobbing 0.2 — motion-sickness saver, no downside.
  • The Mouse 5 combo (host-approved): bind EFT's Hold Breath AND FOV Fix's Zoom key both to Mouse 5 — one thumb press steadies your aim and eagle-eyes distant targets at the same time. Feels amazing, costs nothing.
  • Rebind in F12 to keys you'll actually press: Dynamic Maps peek, Amands' Sense pulse. Fika defaults: ping ; · extract F8 · give up when downed End.

Mod-by-mod F12 tweaks worth making

ModWorth touching
Fontaine's FOV FixSet the Zoom keybind (→ Mouse 5 combo above). Per-scope ADS FOV is there if a specific optic feels off — otherwise defaults are good.
Amands' SenseTune range and highlight duration to taste. Keep range modest (~30 m) — at high values it stops being "sense" and starts being a wallhack, and kills the loot-hunt fun.
Smajlec's LightsThe look of the whole game lives here. Too bright / washed out → lower intensity or switch preset. Do this before touching PostFX or monitor settings.
AO ReplacerTry its GTAO presets — "shaded corners" depth for ~free. If a map feels too dark indoors, drop it a notch instead of raising brightness.
Dynamic MapsBind the peek key (glance while moving), enable auto-center on player. Quest + extract markers are already on.
Borkel's NVGRaise Gain a touch if night raids feel too dark; add a little Noise for realism.
UI FixesMostly great out of the box. Favorites to know exist: swap-in-place drag, autostack on transfer, mousewheel scope zoom, load-ammo-preset. (Two gameplay toggles are host-enforced in raid — that's intended.)
HollywoodFXIf FPS dips during big firefights, lower its effect intensity/density — it's the priciest visual mod in heavy combat.
Quest TrackerBind the overlay toggle; set auto-hide so it's not permanently on screen.
All F12 mod settings are personal — they only affect your game. Tune freely; you can't break the server from there. The one exception you'll notice: a couple of UI Fixes gameplay toggles are enforced by the raid host for fairness.
